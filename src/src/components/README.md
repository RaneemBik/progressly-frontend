# frontend/src/src/components

Reusable React components, grouped by feature area.

| Folder | Contents |
|---|---|
| `ui/` | Generic UI primitives: Button, Input, Modal, Toast |
| `layout/` | App-wide layout: Navbar, Sidebar, Footer |
| `landing/` | Landing page sections: ThreeBackground, Features |
| `dashboard/` | Dashboard-specific: ProjectCard, StatsCards, PendingInvitations |
| `project/` | Project page: TaskBoard, TaskCard, TaskColumn, TaskForm, DependencyGraph, MemberList, InviteMemberModal, ProjectInvitations |
