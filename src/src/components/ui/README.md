# frontend/src/src/components/ui

Generic, reusable UI primitive components with no business logic.

| File | Purpose |
|---|---|
| `Button.tsx` | Button with variants (primary/secondary/outline/ghost/danger), sizes, loading state |
| `Input.tsx` | Form input with optional label, icon, and error message |
| `Modal.tsx` | Animated overlay modal dialog with Escape-key and backdrop-click close |
| `Toast.tsx` | Toast notification system (success/error/warning/info) + ConfirmDialog |
