# frontend/src/src/components/project

Components used on the ProjectPage.

| File | Purpose |
|---|---|
| `TaskBoard.tsx` | Kanban board container; manages drag-drop state and API calls |
| `TaskColumn.tsx` | Single Kanban column with HTML5 drag-and-drop and drop validation |
| `TaskCard.tsx` | Individual task card; shows blocked state, parent status, assignee |
| `TaskForm.tsx` | Create/edit task modal with dependency picker |
| `DependencyGraph.tsx` | Interactive SVG graph of task dependencies with animated edges |
| `MemberList.tsx` | Slide-in panel listing team members with roles |
| `InviteMemberModal.tsx` | Modal to invite a team member by email |
| `ProjectInvitations.tsx` | Paginated list of all project invitations (owner/admin only) |
